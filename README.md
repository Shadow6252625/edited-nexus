## Quick Start

### Local Development

Simply open `index.html` in your web browser:

```bash
# Windows
start index.html

# macOS
open index.html

# Linux
xdg-open index.html
```

### GitHub Pages Deployment

1. Push this repository to GitHub
2. Go to Settings â†’ Pages
3. Select the branch and set source to `/ (root)`
4. Your site will be live at `https://yourusername.github.io/nexus/`

- **Netlify** - Drag and drop `index.html`
- **Vercel** - Upload `index.html`
- **Cloudflare Pages** - Upload `index.html`
- **Any static hosting** - Just upload `index.html`

The application supports:
